👉 메추리알 장조림

메추리알 장조림, 간단한 밑반찬으로 이만한게 없쥬?

>재료 목록:

재료 용량
메추리알 1봉지
물 1/2컵
간장 6큰술
설탕 2큰술
올리고당
미림

조리 순서:

  1. 깐 메추리알을 채에 받쳐서 여러번 깨끗히 씻어서 준비해주세요.
  2. 냄비에 메추리알과 분량의 양념을 넣어주세요.
  3. 양념장에 물은 머그컵으로 반정도 넣어주고 간장,설탕,미림,올리고당을 넣어주세요.
  4. 분량의 양념들을 더 넣고 센불로 한번 팔팔 끓여주시고,끓기 시작하면 중불로 국물이 거의 없어질 정도로 졸여주시면 돼요.
  5. 바글바글~졸여주시면 국물이 거의 안남았을쯤에 거품이 보글보글 올라오기 시작할 때 불을 꺼주시면 된답니다.

👉 계란참치주먹밥

계란참치주먹밥 소풍도시락 간단요리!

>재료 목록:

재료 용량
밥 1공기
계란 2개
후추 1/2스푼
참치 한캔
소금 3꼬집
대파 1개

조리 순서:

  1. 대파는 얇게 채썰고,계란은 잘 풀어 준비해 줍니다.
  2. 그릇에 밥,기름 버린 참치, 대파를 넣고 소금, 후추로 간 한뒤 동그랗게 뭉쳐줍니다.
  3. 뭉친밥을 계란물에 입힌 후 팬에 식용유를 붓고 노릇하게 구워주면 완성입니다.

👉 소세지야채볶음

소세지야채볶음 황금레시피

>재료 목록:

재료 용량
비엔나소세지 1봉지
양파 1/3
파프리카
당근
식용유 1번
케찹 2T
고추장 1T
굴소스
올리고당
다진마늘 1/3T

조리 순서:

  1. 야채를 먹기 좋은 크기로 잘라주세요.
  2. 비엔나 소세지에 칼집을 내어 끓는 물에 데쳐주세요.
  3. 양념장을 만들어주세요.
    [양념장 제작방법!]
    (케첩2스푼, 고추장1스푼, 굴소스1스푼, 올리고당1스푼, 다진마늘1/3스푼)
  4. 식용유를 팬에 두르고, 손질해둔 야채는 중약불로 볶아주세요.
  5. 야채가 어느정도 익었다면 소세지를 넣고 2~3분간 볶아주세요.
  6. 미리 만들어둔 양념장을 넣고 약불로 3분간 볶아주세요.
